# Responsive Navbar



A modern and responsive navigation bar built using **HTML, CSS, and JavaScript**. The project features a clean user interface with responsive design, search functionality, login modal, and a mobile-friendly hamburger menu.



## Features



* Responsive navigation bar

* Search modal popup

* Login modal popup

* Hamburger menu for mobile devices

* Smooth hover effects

* Responsive layout using Media Queries

* Interactive UI with JavaScript

* Modern and clean design



## Technologies Used



* HTML5

* CSS3

* JavaScript (ES6)



## Functionality



### Search Modal



* Opens when the search icon is clicked.

* Closes using the close button or by clicking outside the modal.



### Login Modal



* Opens when the user icon is clicked.

* Contains email and password input fields.

* Closes using the close button or by clicking outside the modal.



### Mobile Navigation



* Displays a hamburger menu on smaller screens.

* Navigation links appear in a dropdown menu when the hamburger icon is clicked.
